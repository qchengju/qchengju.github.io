picgo+github 配置个人图床

发表于2025-09-09|更新于2025-09-19|环境&配置
|阅读量:

使用场景（需求）

前端使用 img 标签时可以直接将 src 换成网络路径直接展示，还有就是最常用的 md 中的图片插入，可以只需要在写的时候写上路径就能在任何地方展示。

配置方法

  1. 注册一个 github 账号，在设置中找到 token 选择 classic token
  2. 将第一个作为仓库勾选上，生成
  3. 记住你的 token 它只会出现这一次，以后你将无法访问到它。
