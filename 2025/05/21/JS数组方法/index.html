js的数组方法

发表于2025-05-21|更新于2025-07-07|js
|阅读量:

介绍

JS 数组是极为灵活的数据结构，应用极为广泛，在这些封装好的方法帮助下，可以大大提高开发效率，和代码可读性。

方法

1. arr.push( )

该方法接收一个或多个参数并将所有的参数按顺序插入数组的末尾。
若将数组作为参数传入，不会将传入的数组拆开而是直接加入到原数组中。

2. arr.forEach(callback )

该方法接收一个回调函数作为参数 ，

3. arr.map(callback)

该方法同样接收一个回调函数作为参数，

4. arr.fliter(callback)

该方法也接收一个回调函数为参

5. arr.findIndex(callback)

该方法接收一个回调函数

6. arr.splice(a,b,c )

该方法接收三个参数，第一个是修改的索引，第二个是将要删除的个数，第三个或更多的元素是在要修改位置插入的，这个方法会原地修改数组。

7. arr.sort(callback)

该方法为所有数组元素进行了排序，按照升序或者降序排列。

8. arr.reverse( )

该方法可以直接将数组的元素反转顺序。

9. arr.shift( )

删除第一个元素并返回该元素。

10. arr.pop( )

删除末尾元素并返回删除的元素。

11. arr.unshift( )

在数组起始处添加元素并返回数组长度。

12. arr.join( )

可以将数组转化为字符串，并将该字符串返回，传入的字符参数会作为里连接符。
参数为字符。

13. arr.concat( )

传入两个数组合成一个数组并反回新数组。

14. arr.slice( )

arr.slice(起始索引，结束索引)
包含开始不包含结束。
可以只写一个参数，截取从该元素开始以后的所有元素。
还可以写负参数，代表截取到倒数第几个元素。

15. arr.includes( )

判断传入的元素是否在数组中。
