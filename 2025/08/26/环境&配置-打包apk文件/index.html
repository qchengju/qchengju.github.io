将 uniapp 项目打包成 安卓手机应用 apk

环境准备

打包安卓应用，首先需要的编辑器有 Android Studio，hbuilderx.
hbuilderx是对 uniapp 应用具有极好的适配。

云打包

通过使用DCloud账号登录后，可以通过 hbx 直接云端打包，打包完成后可以根据提示路径下载apk文件。本文的重点是介绍，本地打包。

本地打包

  1. 使用 hbuilderx 打开项目，并选择 发行 -> 本地打包 -> 生成本地打包资源 。这一步通常是生成了一个 UNI*** 的文件夹。这个文件夹里就是本地的打包资源。
  2. 打开 Android Studio ，新建项目，选择导入项目，选择 android/app 文件夹。
  3. 将证书文件复制到 android/app 文件夹下。 android 证书通常是 .keystore 或者 .jks 文件。
  4. 在这个目录下 找到 build.gradle 文件，配置证书路径密码。
  5. 将android 文件夹导入到 android studio 中。
  6. 点左侧的菜单栏选择 Build-> Generate App Bundles or APKs -> Generate APKs

others

！配置 appid 在 android/app/assets/data/dcloud_control.xml 文件中 UNI*** 格式的appid 配置。

！需要在dcloud 账号中申请appid,注册并且完成认证，和创建项目。才能完成云打包。
