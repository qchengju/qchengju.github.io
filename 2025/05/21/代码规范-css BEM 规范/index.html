CSS BEM 规范详解

BEM（Block, Element, Modifier）是一种结构化的 CSS 命名方法，由 Yandex 团队提出，用于创建可复用、可维护的前端组件。核心思想是将 UI 拆分为独立的块，通过清晰的命名规则建立层级关系。

BEM 核心概念

1. Block（块）

  • 定义：页面中独立的、有意义的组件（如 header, card, menu）。
  • 命名：使用单个单词或连字符连接（如 user-card, search-form）。

2. Element（元素）

  • 定义：块的组成部分，不能独立于块存在（如 navbar__link, card__title）。
  • 命名：使用双下划线 __ 连接块名和元素名（如 block__element）。

3. Modifier（修饰符）

  • 定义：改变块或元素的外观、状态或行为（如 button--primary, card--disabled）。
  • 命名：使用双连字符 -- 连接（如 block--modifier 或 block__element--modifier）。

BEM 命名规则

  1. 使用连字符 - 分隔单词（如 user-card）。
  2. 元素用双下划线 __ 连接块（如 user-card__avatar）。
  3. 修饰符用双连字符 -- 连接（如 user-card--highlighted）。
  4. 避免嵌套元素超过两层（如 block__elem1__elem2 不推荐）。
